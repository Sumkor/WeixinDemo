# WeixinDemo

## Tip

由于微信公众平台于2015/8/15新增图文内容中图片上传接口，并过滤外链图片，需要发送以下POST请求，获得上传图片的URL

```
https://api.weixin.qq.com/cgi-bin/media/uploadimg?access_token=ACCESS_TOKEN
```
